Purpose:

Quick reference for common EAS CLI commands.

Build:

eas build --platform ios
eas build --platform android
eas build --platform all
eas build --platform ios --profile production
eas build:list

Submit:

eas submit --platform ios
eas submit --platform android
eas submit:list

Credentials:

eas credentials
eas device:create

Project:

eas update
eas branch:list
eas env:list
eas env:create

Diagnostics:

npx expo-doctor
eas whoami
